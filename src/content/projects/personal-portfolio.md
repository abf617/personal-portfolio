# Allen Aronis Portfolio

This personal portfolio website showcases a unique 90s cyberpunk aesthetic combined with modern web technologies to create an immersive terminal experience.

## Features

- **Matrix Background**: Canvas-based falling code animation for authentic Matrix vibes
- **CRT Effects**: Scanlines, phosphor glow, and chromatic aberration for retro monitor feel
- **Terminal UI**: Navigation and interactions styled as command-line interface
- **Typing Animations**: Dynamic text that types out like terminal commands
- **Glitch Transitions**: RGB channel separation effects on hover and transitions
- **Accessibility**: Full support for prefers-reduced-motion

## Tech Stack

Built with cutting-edge tools:

- Astro 5.x for static site generation
- React for interactive components
- Tailwind CSS v4 for styling
- TypeScript for type safety

## Performance

Despite heavy visual effects, the site maintains excellent performance:

- 60fps Matrix animation using canvas
- Lazy-loaded React components using Astro Islands
- Optimized bundle size with code splitting
- Accessibility-first approach

## Result

A unique portfolio that stands out while maintaining excellent UX and performance metrics.
